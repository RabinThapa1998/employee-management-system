# WEBO

Full Stack Employee Management System

## Tech Stack

### Frontend

Vite ⚡ React JS ⚡ AntDesign ⚡ React Router ⚡ React Query ⚡

### Backend

Node ⚡ Express ⚡ MongoDB

## Installation

clone the repository

```bash
 git clone https://github.com/RabinThapa1998/github_repo_search.git
```

client

```bash
cd client
pnpm install
pnpm dev
--->localhost:5173
```

server

```bash
cd server
pnpm install
pnpm dev
--->localhost:8000
```
